# Portless.NET v3.0 Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Add a plugin system and request inspector to make Portless.NET extensible and transparent.

**Architecture:** Plugin SDK as a shared library loaded at runtime via AssemblyLoadContext. Inspector as a capturing middleware + ring buffer + Spectre.Console TUI + WebSocket API. Both integrate into the existing YARP middleware pipeline in Program.cs.

**Tech Stack:** .NET 10, YARP 2.3.0, Spectre.Console 0.53.1, System.Text.Json, WebSockets

```csharp
namespace Portless.Plugin.SDK;

/// <summary>
/// Base class for all Portless plugins.
/// Override the hooks you need - all are optional.
/// </summary>
public abstract class PortlessPlugin
{
    /// <summary>Plugin name (must be unique).</summary>
    public abstract string Name { get; }
    
    /// <summary>Plugin version.</summary>
    public abstract string Version { get; }

    // Lifecycle hooks
    public virtual Task OnLoadAsync(IPluginContext context) => Task.CompletedTask;
    public virtual Task OnUnloadAsync() => Task.CompletedTask;

    // Request pipeline hooks - return non-null to short-circuit the pipeline
    public virtual Task<ProxyResult?> BeforeProxyAsync(ProxyContext context) => Task.FromResult<ProxyResult?>(null);
    public virtual Task AfterProxyAsync(ProxyContext context, ProxyResult result) => Task.CompletedTask;

    // Route management hooks
    public virtual Task OnRouteAddedAsync(RouteInfo route) => Task.CompletedTask;
    public virtual Task OnRouteRemovedAsync(RouteInfo route) => Task.CompletedTask;

    // Error hooks - return non-null to override error response
    public virtual Task<ErrorResponse?> OnErrorAsync(ErrorContext context) => Task.FromResult<ErrorResponse?>(null);
}
```

### Task 3: Integrate plugin hooks into proxy pipeline

**Objective:** Add plugin hook calls to Program.cs middleware pipeline.

**Files:**
- Modify: `Portless.Proxy/Program.cs`
- Modify: `Portless.Proxy/Portless.Proxy.csproj` (add reference to Plugin.SDK)
- Modify: `Portless.Core/Extensions/ServiceCollectionExtensions.cs`

**Approach:**
1. Register `IPluginLoader` as singleton in DI
2. Load plugins on startup (after YARP config)
3. Add `app.UseMiddleware<PluginMiddleware>()` before `app.MapReverseProxy()`
4. PluginMiddleware calls `FireBeforeProxyAsync` before forwarding, `FireAfterProxyAsync` after
5. Error page middleware calls `FireErrorAsync` before generating default error page
6. Route add/remove API endpoints call `FireRouteAddedAsync/FireRouteRemovedAsync`

**PluginMiddleware.cs** (new file in Portless.Proxy/):

```csharp
public class PluginMiddleware
{
    private readonly RequestDelegate _next;
    private readonly IPluginLoader _pluginLoader;
    private readonly ILogger<PluginMiddleware> _logger;

    public async Task InvokeAsync(HttpContext context)
    {
        var pluginLoader = context.RequestServices.GetRequiredService<IPluginLoader>();
        
        var proxyContext = new ProxyContext
        {
            Hostname = context.Request.Host.Host,
            Path = context.Request.Path,
            Method = context.Request.Method,
            Scheme = context.Request.Scheme,
            Headers = context.Request.Headers.ToDictionary(h => h.Key, h => h.Value.ToString()),
            CancellationToken = context.RequestAborted
        };

        var sw = Stopwatch.StartNew();
        
        // Before hook
        var shortCircuit = await pluginLoader.FireBeforeProxyAsync(proxyContext);
        if (shortCircuit != null)
        {
            context.Response.StatusCode = shortCircuit.StatusCode;
            foreach (var header in shortCircuit.ResponseHeaders)
                context.Response.Headers[header.Key] = header.Value;
            if (shortCircuit.ResponseBody != null)
                await context.Response.WriteAsync(shortCircuit.ResponseBody);
            return;
        }

        await _next(context);
        
        sw.Stop();
        
        // After hook
        var result = new ProxyResult
        {
            StatusCode = context.Response.StatusCode,
            DurationMs = sw.ElapsedMilliseconds,
            ResponseHeaders = context.Response.Headers.ToDictionary(h => h.Key, h => h.Value.ToString())
        };
        await pluginLoader.FireAfterProxyAsync(proxyContext, result);
    }
}
```

### Task 8: Create InspectorMiddleware

**Objective:** Middleware that captures every proxied request/response into the ring buffer.

**Files:**
- Create: `Portless.Proxy/InspectorMiddleware.cs`

**Approach:**
- Intercept the response body stream (same pattern as error page middleware)
- Record timing with Stopwatch
- Capture request/response headers
- Store body only if Content-Type is text-based (json, html, xml, text) and < 1MB
- Must be non-blocking (fire-and-forget capture into buffer)
- Must be registered AFTER PluginMiddleware but BEFORE MapReverseProxy

```csharp
public class InspectorMiddleware
{
    private readonly RequestDelegate _next;
    
    public async Task InvokeAsync(HttpContext context)
    {
        var inspector = context.RequestServices.GetRequiredService<IRequestInspector>();
        
        // Don't capture internal API calls
        if (context.Request.Path.StartsWithSegments("/api/v1") ||
            context.Request.Path.StartsWithSegments("/_dashboard"))
        {
            await _next(context);
            return;
        }
        
        var sw = Stopwatch.StartNew();
        var originalBody = context.Response.Body;
        using var captureStream = new MemoryStream();
        context.Response.Body = captureStream;
        
        // Capture request body if text-based
        string? requestBody = null;
        if (context.Request.ContentLength < 1_000_000 &&
            context.Request.ContentType?.Contains("json") == true ||
            context.Request.ContentType?.Contains("text") == true)
        {
            using var reader = new StreamReader(context.Request.Body);
            requestBody = await reader.ReadToEndAsync();
            context.Request.Body = new MemoryStream(Encoding.UTF8.GetBytes(requestBody));
        }
        
        await _next(context);
        sw.Stop();
        
        // Capture response body if text-based
        string? responseBody = null;
        captureStream.Seek(0, SeekOrigin.Begin);
        var contentType = context.Response.ContentType ?? "";
        if ((contentType.Contains("json") || contentType.Contains("text") || 
             contentType.Contains("html") || contentType.Contains("xml")) &&
            captureStream.Length < 1_000_000)
        {
            using var reader = new StreamReader(captureStream);
            responseBody = await reader.ReadToEndAsync();
        }
        
        // Copy response back
        captureStream.Seek(0, SeekOrigin.Begin);
        context.Response.Body = originalBody;
        await captureStream.CopyToAsync(originalBody);
        
        // Record
        inspector.Capture(new CapturedRequest
        {
            Method = context.Request.Method,
            Hostname = context.Request.Host.Host,
            Path = context.Request.Path + context.Request.QueryString,
            Scheme = context.Request.Scheme,
            RequestHeaders = context.Request.Headers.ToDictionary(h => h.Key, h => h.Value.ToString()),
            RequestBody = requestBody,
            StatusCode = context.Response.StatusCode,
            ResponseHeaders = context.Response.Headers.ToDictionary(h => h.Key, h => h.Value.ToString()),
            ResponseBody = responseBody,
            DurationMs = sw.ElapsedMilliseconds
        });
    }
}
```
