  <files>Portless.Cli/Services/IProxyProcessManager.cs, Portless.Cli/Services/ProxyProcessManager.cs</files>
  <action>
Create IProxyProcessManager interface with:
- StartAsync(int port) - Starts proxy as detached process, returns Task
- StopAsync() - Stops running proxy process, returns Task
- IsRunningAsync() - Returns bool if proxy is running
- GetStatusAsync() - Returns (bool isRunning, int? port, int? pid)

Implement ProxyProcessManager:
1. **PID file management:**
   - Use StateDirectoryProvider.GetStateDirectory() to get state dir
   - PID file path: {stateDir}/proxy.pid
   - Write PID immediately after Process.Start succeeds
   - Delete PID file on StopAsync

2. **StartAsync implementation:**
   - Check if already running via IsRunningAsync
   - If running, throw InvalidOperationException with message "Proxy is already running. Use 'portless proxy stop' first"
   - Build path to Portless.Proxy.csproj using Path.Combine with relative paths from Assembly.Location
   - Create ProcessStartInfo with:
     * FileName = "dotnet"
     * Arguments = $"run --project \"{proxyProjectPath}\" --urls http://*:{port}"
     * UseShellExecute = true (for detached execution)
     * CreateNoWindow = true
     * WindowStyle = ProcessWindowStyle.Hidden
     * Environment["PORTLESS_PORT"] = port.ToString()
     * Environment["DOTNET_MODIFIABLE_ASSEMBLIES"] = "debug" (hot reload support)
   - Call Process.Start(startInfo)
   - If process is null, throw InvalidOperationException("Failed to start proxy process")
   - Write process.Id to proxy.pid file
   - await Task.Delay(500) to allow startup verification
   - Return without waiting for process to complete (detached)

3. **StopAsync implementation:**
   - Read PID from proxy.pid file
   - If file doesn't exist, throw InvalidOperationException("Proxy is not running")
   - Parse PID, call Process.GetProcessById(pid)
   - Call process.Kill(entireProcessTree: true) for clean shutdown
   - Delete proxy.pid file
   - Handle ArgumentException (PID doesn't exist) - delete stale PID file and return

4. **IsRunningAsync implementation:**
   - Check if proxy.pid exists
   - If not, return false
   - Parse PID from file
   - Call Process.GetProcessById(pid) in try/catch
   - Return !process.HasExited if found, false if ArgumentException

5. **GetStatusAsync implementation:**
   - If not running, return (false, null, null)
   - Parse PID and extract port from environment (may need to store port in separate file for retrieval)
   - For now, return (true, defaultPort 1355, pid) - port tracking can be enhanced later

DO NOT:
- Use WaitForExit or WaitForExitAsync (blocks detached execution)
- Create windows or show UI (CreateNoWindow must be true)
- Use ShellExecute = false (breaks detached execution on Windows)

   using Spectre.Console.Cli;

   public class TypeRegistrar : ITypeRegistrar
   {
       private readonly IServiceCollection _services;

       public TypeRegistrar(IServiceCollection services)
       {
           _services = services;
       }

       public ITypeResolver Build()
       {
           return new TypeResolver(_services.BuildServiceProvider());
       }

       public void Register(Type service, Type implementation)
       {
           _services.AddSingleton(service, implementation);
       }

       public void RegisterInstance(Type service, object implementation)
       {
           _services.AddSingleton(service, implementation);
       }

       public void RegisterLazy(Type service, Func<object> factory)
       {
           _services.AddSingleton(service, _ => factory());
       }
   }

   **Portless.Cli/DependencyInjection/TypeResolver.cs:**
   ```csharp
   using Microsoft.Extensions.DependencyInjection;
   using Spectre.Console.Cli;

   public class TypeResolver : ITypeResolver
   {
       private readonly IServiceProvider _provider;

       public TypeResolver(IServiceProvider provider)
       {
           _provider = provider;
       }

       public object? Resolve(Type? type)
       {
           if (type == null) return null;
           return _provider.GetService(type);
       }

       public void Dispose()
       {
           if (_provider is IDisposable disposable)
           {
               disposable.Dispose();
           }
       }
   }
   ```

   These implementations bridge Spectre.Console.Cli's DI abstraction with .NET's ServiceCollection.
