<task type="auto">
  <name>Implement certificate manager orchestration service</name>
  <files>Portless.Core/Services/CertificateManager.cs</files>
  <action>
    Create CertificateManager class implementing ICertificateManager with:

    **Constructor:**
    - Inject ICertificateService, ICertificateStorageService, ILogger<CertificateManager>
    - Define CertificateGenerationOptions with defaults (5-year validity, 4096-bit CA key, 2048-bit server key)

    **EnsureCertificatesAsync method:**
    - If forceRegeneration is true, call RegenerateCertificatesAsync and return CertificateStatus with IsValid=true, Message="Certificates regenerated"
    - Check if certificate files exist via storageService.CertificateFilesExistAsync()
    - If files don't exist: Generate CA via certificateService.GenerateCertificateAuthorityAsync(), save via storageService.SaveCertificateAuthorityAsync(), generate server cert via certificateService.GenerateWildcardCertificateAsync(), save via storageService.SaveServerCertificateAsync(), create and save metadata via storageService.SaveCertificateMetadataAsync(), return status with IsValid=true, Message="Certificates generated"
    - If files exist: Load CA via storageService.LoadCertificateAuthorityAsync(), validate certificate (not null, HasPrivateKey, not expired, not before valid), if validation fails log warning and regenerate, return status with IsValid=true or IsValid=false based on validation
    - Use try-catch for CryptographicException to detect corrupted PFX files

    **GetCertificateAuthorityAsync method:**
    - Call EnsureCertificatesAsync() to guarantee valid certificate exists
    - Load and return CA certificate from storageService.LoadCertificateAuthorityAsync()
    - Throw InvalidOperationException if certificate is null after ensure (should never happen)

    **GetServerCertificateAsync method:**
    - Call EnsureCertificatesAsync() to guarantee valid certificate exists
    - Load and return server certificate from storageService.LoadServerCertificateAsync()
    - Throw InvalidOperationException if certificate is null after ensure

    **GetCertificateStatusAsync method:**
    - Load metadata from storageService.LoadCertificateMetadataAsync()
    - If metadata is null, load CA certificate and create status from certificate properties
    - Check expiration: if NotAfter < UtcNow → IsExpired=true, if NotAfter < UtcNow.AddDays(30) → IsExpiringSoon=true
    - Check certificate validity: load CA via storageService.LoadCertificateAuthorityAsync(), set IsCorrupted=true if load fails or HasPrivateKey is false
    - Calculate NeedsRegeneration = IsCorrupted || IsExpired || IsExpiringSoon
    - Return CertificateStatus with all properties populated

    **RegenerateCertificatesAsync method:**
    - Generate new CA via certificateService.GenerateCertificateAuthorityAsync()
    - Generate new server certificate via certificateService.GenerateWildcardCertificateAsync()
    - Create CertificateInfo with SHA-256 thumbprint, ISO 8601 dates, Unix timestamps
    - Save all via storageService methods
    - Log information message about regeneration

    **Validation helper method (private):**
    - ValidateCertificate(X509Certificate2? cert) returns (bool IsValid, string? Reason)
    - Checks: cert != null, cert.HasPrivateKey, UtcNow >= cert.NotBefore, UtcNow <= cert.NotAfter
    - Returns tuple with validation result and reason for invalid state

    **Key patterns from user context:**
    - "Certificados existentes: Reutilizar siempre si existen, sin preguntar" → Always reuse existing valid certificates
    - "Certificados corruptos: Regenerar automáticamente con warning al usuario" → Auto-regenerate on corruption with log warning
    - "Primera generación: Preguntar al usuario la primera vez" → Deferred to Phase 14 CLI integration (current plan auto-generates)

    Add comprehensive XML documentation comments.
  </action>

  </verify>
