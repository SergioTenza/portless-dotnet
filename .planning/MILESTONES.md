# Milestones

## v1.0 MVP (Shipped: 2026-02-21)

**Phases completed:** 7 phases, 20 plans

**Key accomplishments:**
- **Phase 01:** HTTP proxy YARP with hostname routing and dynamic configuration
- **Phase 02:** Route persistence with file locking, hot-reload, and automatic cleanup
- **Phase 03:** CLI commands (proxy start/stop/status, list, run) with Spectre.Console
- **Phase 04:** Automatic port allocation (4000-4999) with conflict detection
- **Phase 05:** Process management with PORT injection and graceful shutdown
- **Phase 06:** Global dotnet tool packaging (761KB) with 4 integration examples and 3,049 lines of documentation
- **Phase 08:** Comprehensive integration test suite (45 tests) covering CLI, proxy, and persistence

**Delivered:**
- Stable `.localhost` URLs for Windows .NET development
- Full CLI experience with colored output and table formatting
- Cross-platform installation scripts (bash/PowerShell)
- Integration examples for WebApi, Blazor, Worker, and Console apps
- Tutorial documentation (migration, new project, microservices, E2E testing)

**Platform:** Windows 10+ (macOS/Linux validation deferred to v1.1)
