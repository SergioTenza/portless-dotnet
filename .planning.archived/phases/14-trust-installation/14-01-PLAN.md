<task type="auto">
  <name>Implement CertificateTrustService with Windows X509Store API</name>
  <files>Portless.Core/Services/CertificateTrustService.cs</files>
  <action>
    Create Windows-specific implementation following these requirements:

    **Class declaration:**
    - Mark with [SupportedOSPlatform("windows")] attribute
    - Implement ICertificateTrustService
    - Inject ILogger&lt;CertificateTrustService&gt; and ICertificateManager dependencies

    **InstallCertificateAuthorityAsync implementation:**
    - Use X509Store(StoreName.Root, StoreLocation.LocalMachine) with OpenFlags.ReadWrite
    - Check if certificate already exists via Find(X509FindType.FindByThumbprint, thumbprint, validOnly: false)
    - If exists: return TrustInstallResult with Success=true, AlreadyInstalled=true
    - If not exists: call store.Add(certificate) and return TrustInstallResult with Success=true
    - Catch CryptographicException with HResult -2146829211 (ERROR_E_ACCESS_DENIED): return TrustInstallResult with RequiresAdmin=true, ErrorMessage="Run as Administrator"
    - Catch other exceptions: return TrustInstallResult with ErrorMessage containing ex.Message

    **GetTrustStatusAsync implementation:**
    - Load CA certificate using ICertificateManager.GetCertificateAuthorityAsync()
    - Open X509Store(StoreName.Root, StoreLocation.LocalMachine) with OpenFlags.ReadOnly
    - Search for certificate via Find(X509FindType.FindByThumbprint, thumbprint, validOnly: true)
    - If Count == 0: return TrustStatus.NotTrusted
    - If found: check certificate.NotAfter &lt; DateTimeOffset.UtcNow.AddDays(30) - if true, return TrustStatus.ExpiringSoon
    - Otherwise: return TrustStatus.Trusted
    - Handle CryptographicException for store access failures

    **UninstallCertificateAuthorityAsync implementation:**
    - Open X509Store(StoreName.Root, StoreLocation.LocalMachine) with OpenFlags.ReadWrite
    - Search for certificate via Find(X509FindType.FindByThumbprint, thumbprint, validOnly: false)
    - If Count == 0: return true (idempotent - already not installed)
    - Call store.Remove(certificates[0]) to remove certificate
    - Return true on success, false on exception

    **IsAdministratorAsync implementation:**
    - Use WindowsIdentity.GetCurrent() to get current identity
    - Create WindowsPrincipal from identity
    - Return principal.IsInRole(WindowsBuiltInRole.Administrator)
    - Catch exceptions (UnauthorizedAccessException, SecurityException) and return false

    **Logging:**
    - Log.LogInformation when certificate is installed successfully
    - Log.LogWarning when certificate already installed
    - Log.LogError when CryptographicException occurs with store access

    **Platform guards:**
    - Add OperatingSystem.IsWindows() check at start of each method
    - Throw PlatformNotSupportedException("Certificate trust installation is Windows-only") if not on Windows
  </action>
