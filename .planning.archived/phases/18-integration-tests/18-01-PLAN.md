5. Create test method `Certificate_Renewal_Generates_New_Certificate()`:

   - Act: Call `await _certManager.RegenerateCertificatesAsync(forceRegeneration: true)`
   - Assert: New certificate has different thumbprint
   - Assert: New certificate `NotAfter` is ~5 years from now
6. Create test method `Certificate_Renewal_Updates_Metadata_File()`:
   - Arrange: Generate certificate, read initial metadata from `cert-info.json`
   - Act: Force renewal, wait for file update
   - Assert: Metadata file exists
   - Assert: New thumbprint in metadata
   - Assert: New expiration date in metadata
   - Use `ICertificateStorageService.LoadCertificateInfoAsync()` to read metadata
7. Create test method `Certificate_Expiration_Detection_Works_Before_30_Days()`:
   - Arrange: Create certificate with custom expiration (mock by checking `IsCertificateNearExpiration` logic)
   - Act: Check `await _certManager.IsCertificateNearExpiration(thresholdDays: 30)`
   - Assert: Returns correct boolean based on certificate age
   - Note: This test verifies the detection logic, not actual time-based expiration (unreliable in tests)
